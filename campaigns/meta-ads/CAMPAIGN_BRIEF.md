## KPIs & Decision Framework

### Target Metrics
| Metric | Target |
|--------|--------|
| CTR | > 1% (good), > 2% (excellent) |
| CPL (Diaspora) | < $15 |
| CPL (Israel) | < 30 NIS |
| VTR (75%+) | Track per variant |
| CPC | Compare across languages (expect FR = cheapest) |

### 24-Hour Decision Matrix
| Result | Action |
|--------|--------|
| CTR > 2% | Double budget |
| CTR 1-2% | Extend 48 hours |
| CTR < 1% | Stop, replace creative |
| CPL < $10 | Scale aggressively |
| Market with 0 leads | Stop, review targeting |

---

## Scale Plan (Week 2+)
1. Kill losers (low CTR variants)
2. Double winners
3. Custom Audiences from website visitors + video viewers (75%+)
4. Lookalike 1% from form submissions
5. Advantage+ after 50+ conversions/week
